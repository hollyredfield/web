Elementos CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño que se utiliza para controlar la presentación y el diseño de un documento HTML. Aquí se describen algunos elementos clave de CSS y sus aplicaciones:

  1. Selector: Un selector en CSS apunta a los elementos HTML a los que se aplicarán los estilos.
    Ejemplo: 
     p { color: blue; }
  2. Propiedad: Define el aspecto que se aplicará al elemento seleccionado.
    Ejemplo: 
     color: red;
  3. Valor: Especifica cómo se aplicará la propiedad al elemento.
    Ejemplo: 
     font-size: 16px;
  4. Color: Permite definir el color del texto o del fondo.
    Ejemplo: 
     color: #336699;
  5. Font: Controla la apariencia del texto, como tipo de letra y tamaño.
    Ejemplo: 
     font-family: 'Arial', sans-serif;
  6. Margin: Establece los márgenes alrededor de un elemento.
    Ejemplo: 
     margin: 10px;
  7. Padding: Define el espacio entre el contenido del elemento y su borde.
    Ejemplo: 
     padding: 15px;
  8. Border: Crea bordes alrededor de los elementos.
    Ejemplo: 
     border: 1px solid #999;
  9. Width: Establece el ancho de un elemento.
    Ejemplo: 
     width: 300px;
  10. Height: Define la altura de un elemento.
    Ejemplo: 
     height: 200px;
  11. Display: Controla cómo se muestra un elemento.
    Ejemplo: 
     display: flex;
  12. Position: Determina la posición de un elemento.
    Ejemplo: 
     position: relative;
  13. Float: Hace que un elemento flote a la izquierda o la derecha.
    Ejemplo: 
     float: left;
  14. Opacity: Controla la opacidad de un elemento.
    Ejemplo: 
     opacity: 0.7;
  15. Transition: Añade efectos de transición a los estilos.
    Ejemplo: 
     transition: background 0.5s ease;
