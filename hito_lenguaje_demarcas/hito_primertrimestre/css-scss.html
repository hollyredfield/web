CSS y SCSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para controlar el aspecto y el formato de un documento HTML. Permite definir estilos como colores, márgenes, fuentes y más, para mejorar la presentación de una página web.

SCSS (Sassy CSS) es una extensión de CSS que agrega características como variables, anidamiento y mixins, facilitando la escritura y mantenimiento del código. SCSS es luego compilado a CSS para su implementación en navegadores.

La diferencia principal entre CSS y SCSS radica en la sintaxis. Mientras que CSS utiliza una sintaxis más estricta, SCSS ofrece una sintaxis más flexible y amigable para el desarrollador.

A continuación, se presentan ejemplos de código que ilustran el uso básico de CSS y SCSS:

CSS

                    
                        body {
                            font-family: 'Arial', sans-serif;
                            background-color: #f2f2f2;
                        }
                        h1 {
                            color: #333;
                        }
                        .container {
                            width: 80%;
                            margin: 0 auto;
                        }

SCSS

                    
                        $font-stack: 'Arial', sans-serif;
                        $main-bg-color: #f2f2f2;
                        body {
                            font-family: $font-stack;
                            background-color: $main-bg-color;
                        }
                        h1 {
                            color: #333;
                        }
                        .container {
                            width: 80%;
                            margin: 0 auto;
                        }
